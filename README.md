# Data Analytics - EDA - TechStore - Rocketseat

Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) utilizando um dataset de vendas de uma empresa fictícia chamada **TechStore**.

A análise busca responder perguntas de negócio, gerar insights e criar visualizações para melhor compreensão dos dados.

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- SQLite
- Matplotlib
