# Data Analytics - Limpeza de Dados - Rocketseat

Este projeto tem como objetivo aplicar técnicas de **tratamento e preparação de dados** em um dataset fictício de clientes da empresa **DataClean Co.**.
O foco é deixar os dados prontos para **análises exploratórias e modelagem futura**, seguindo boas práticas utilizadas no dia a dia de um Analista de Dados.

## 🎯 Objetivos

- Realizar limpeza e tratamento de dados
- Identificar e corrigir inconsistências
- Preparar o dataset para análises futuras
- Aplicar técnicas como:
  - Tratamento de valores ausentes
  - Remoção de duplicatas
  - Detecção de outliers
  - Encoding de variáveis categóricas
  - Manipulação de datas

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## 📊 Principais Decisões

- Uso da **mediana** para imputação de valores ausentes
- Remoção seletiva de outliers com base em análise estatística
- Criação de identificador único para resolver inconsistências
- Transformação de variáveis categóricas para uso analítico
