# Data Analytics - Analise-de-Delivery - Rocketseat

Este projeto tem como objetivo realizar uma análise completa de um dataset de pedidos, aplicando técnicas de **Data Analytics** como exploração de dados, tratamento, criação de métricas e geração de insights de negócio.

A análise foi desenvolvida utilizando **Python**, com foco em bibliotecas como **Pandas, NumPy e Matplotlib**.

## 🎯 Objetivos

- Entender a estrutura dos dados
- Tratar valores ausentes
- Criar novas variáveis relevantes
- Realizar agregações e análises por produto
- Analisar comportamento temporal das vendas
- Integrar diferentes fontes de dados
- Gerar KPIs de negócio
- Extrair insights estratégicos

## 🧰 Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## 📊 Insights

- Produtos mais vendidos nem sempre são os mais lucrativos\
- Categoria Salgados gera maior receita\
- Variações ao longo do tempo nas vendas
