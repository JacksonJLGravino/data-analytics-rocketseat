# Lista de Compra - Data Analytics - Rocketseat

Um mini sistema de lista de compras interativo no terminal, desenvolvido em Python, com foco em praticar conceitos fundamentais de lógica de programação, estruturas de dados e organização de código.

### 📌 Sobre o projeto

Este projeto foi desenvolvido como desafio prático do curso de Data Analytics da Rocketseat, com o objetivo de aplicar conceitos básicos de programação utilizando Python.

O sistema permite gerenciar uma lista de compras diretamente pelo terminal, utilizando um menu interativo e armazenamento em memória.

### 🚀 Funcionalidades

- ➕ Adicionar item à lista
- 📋 Listar todos os itens
- ✅ Marcar item como comprado
- ❌ Remover item da lista
- 🔁 Menu interativo em loop

### 🛠️ Tecnologias utilizadas

- Python
- Jupyter Notebook
